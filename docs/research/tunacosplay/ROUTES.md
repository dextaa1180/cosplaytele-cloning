# Route Mapping — Tunacosplay

## Homepage
- **URL**: https://cosplaytele.com/
- **Route**: `/` → `src/app/page.tsx`
- **Purpose**: Grid/list view of cosplay posts

## Detail Page (Dynamic)
- **URL Pattern**: https://cosplaytele.com/[slug]/
- **Route**: `/[slug]/page.tsx` → `src/app/[slug]/page.tsx`
- **Purpose**: Individual cosplay post detail

## Scope
- **For this clone**: Build homepage structure (grid layout, post cards, navigation)
- **Content**: Will be replaced by Ruly with custom content
- **Database**: Not required; static data only
- **Detail pages**: Structure ready (not fully built in Phase 1)
